Brand
Identity:

Problem:
Creating an identity for the font family ‘Work Sans’, highlighting and communicating the typeface’s inherent qualities.

Solution:
The visual language and tone of voice communicates Work Sans’ flexible, yet powerful abilities at a wide variety of applications. Taking influence from corporate information manuals and German industrial philosophy. The font family is displayed as a hard-working font.

Typography:

Problem:
Designing and constructing a type specimen that displays the Work Sans Font Family, using it’s brand identity to communicate. Creating two type systems, one for editorial purpouses and one for display purpouses.

Solution:
The 16 page, saddle bound, Type specimen created displays the Work Sans typeface using the brand identity created. A rigid grid system, paired with a strict editorial type system create a serious tone throughout.
